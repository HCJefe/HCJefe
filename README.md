### About

Founder & CEO of **[Brainstorm AGI](https://brainstormagi.com)** — building AI-powered automation systems for industries that still run on paper, phone calls, and spreadsheets. Based in **Yuma, Arizona**.

Also the founder of **SYNERGY HomeCare of Yuma** — five locations, 250+ employees, $7M+ revenue, serving 300+ veteran and senior clients weekly through the VA Community Care Network.

### What I'm Building

**AI Operations Platform** — Multi-tenant agent system for home care agencies. Scheduling, caregiver management, compliance, and workflow automation powered by Claude AI. Voice, SMS, email, and knowledge base agents that handle the work humans shouldn't have to.

**Enterprise Infrastructure** — CI/CD pipelines with AI-powered code review (CodeRabbit), SAST scanning, and automated deployments on AWS ECS Fargate.
